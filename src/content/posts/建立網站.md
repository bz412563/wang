如何使用 fuwari[https://github.com/saicaca/fuwari/blob/main/docs/README.zh-CN.md ]

1. 下載 pnpm  

Using PowerShell:

```
Invoke-WebRequest https://get.pnpm.io/install.ps1 -UseBasicParsing | Invoke-Expression
```

2. 安裝 node.js
```
https://nodejs.org/zh-tw
```

3. 啟用 pnpm
重啟PowerShell後 在 PowerShell 內 打上
```
corepack enable
```
```
corepack prepare pnpm@latest --activate
```
4. 把檔案拉下來後 在檔案內使用CMD 
```
pnpm install
```

5. pnpm dev 後看看能不能正常開啟網站

```
pnpm dev
```

6. 新頁面

```
pnpm new-post <標題>
```

剩下就可以開始著手更改自己的內容了，上傳到自己網站是之後的事情。
